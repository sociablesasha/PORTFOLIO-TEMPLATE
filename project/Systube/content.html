    • Performance Issue

    1Page, 2Page 등의 육안상의 Paging 기능을 구현하지 않았습니다.

    따라서 로딩과 동시에 많은 Card를 불러와야만 했습니다.

    Card는 간단한 글, 좋아요, 싫어요, Youtube 영상을 포함하고 있었습니다.

    많은 Youtube 영상을 불러올 때, 오랜 시간이 소요됐기에 해결해야만 했습니다.

    Solution

    Youtube Option 설정 : ?controls=2

    Logic 응용 : Paging과 Infinite scroll

    Code 변경 : setTimeout()를 통한 작업 처리 순서 변경

    Result

    6s -> 1.7s
